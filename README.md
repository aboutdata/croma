## Croma #

Lib for getting useful colors from an image.

### How do I get set up? ###

TestColorPicker.java can be used to test.

###Test
Run:

     java -jar run-croma.jar <Image Path> <noOfColors> <algo (0 or 1)> 
     
  It will generate a html file with same name in image dir.
